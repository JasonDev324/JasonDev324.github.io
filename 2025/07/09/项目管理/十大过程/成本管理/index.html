是什么？

项目成本管理过程是关注资源所需的成本和决策对产品的成本影响

有什么？

1. 规划成本管理

理解

  • 确定如何估算、预算、监控、管理、控制项目成本。
  • 成本类型
  1. 机会成本：因选择失去的潜在收益。
  2. 沉没成本：过去发生，未来不能改变的成本。
  3. 直接成本：一个项目工作中用到的资源成本
  4. 间接成本：多个项目都有的成本，例如租金、税费、培训费部分金额

2.估算成本

3.制定预算

4.控制成本

成本管理失控的原因

  1. 制度不全
  2. 管理不当
  3. 技术不行（制约）
  4. 认识不足
  5. 方法问题

怎么记？

储备分析

  • 管理储备和应急储备的区别

    • 应急储备是已知的风险，是成本基准里面的，属于预算。
    • 管理储备是未知风险，不在成本基准里，需要审批。
    • 示例：规划吃席花费，喝酒无节制，是应急储备。喝酒在别人的车里吐了，洗车花费150，这是管理储备。

  • 成本基准
    是已批准的成本支出计划

规划成本管理的ITO？

–

一群乌龟专家开会分析在计划章程上画出了成本管理计划。

解析：

龟画：规划

I：计划章程

T：专家开会分析

O：成本管理计划

PS:规划过程一般都是输入项目章程和项目管理计划，然后单独输出一个过程计划

==

估算成本的ITO？

–

项目经理通过四种估算系统计划文件上得出了成本估算和估算依据。

解析：

四种估算：自下而上估算、三点估算、参数估算、类比估算

系统：项目信息管理系统

I：计划文件

T：四种系统

O：成本估算-估算依据

==
